# 構造計算アプリ Pro (Beam Analysis Pro) v21.30

ブラウザ上で動作する、高機能な連続梁の構造解析・断面力算定ツールです。
三連モーメントの定理に基づき、不連続点やモーメント荷重を考慮した厳密解をリアルタイムで算出・可視化します。



## 🌟 特徴

- **PC向け最適化レイアウト**: ヘッダーを固定し、入力パネルと結果パネルを独立してスクロール可能。
- **高度な解析エンジン**: 集中荷重、分布荷重、台形荷重に加え、集中モーメント荷重に対応。
- **充実の断面データベース**:
  - **S造**: H形鋼、溝形鋼、C形鋼、山形鋼、鋼矢板、角形鋼管に対応。
  - **RC造**: 矩形断面の剛性計算。
  - **任意断面**: $I, Z, E, A$ の直接入力が可能。
- **レポート・データ連携**:
  - A4サイズに最適化された計算書出力（印刷・PDF保存）。
  - CSV形式による入力データの保存・読込および解析結果のエクスポート。

## 📖 計算理論の根拠

本アプリは、変位法（たわみ角法）の基礎となる**一般化された三連モーメントの定理**を用いて解析を行っています。

- **断面力**: 支点モーメントを未知数とする連立方程式から解析解を導出。
- **精度**: 近似解ではなく、梁理論の微分方程式に基づく厳密解を求めているため、市販の有限要素法（FEM）ソフトと理論上一致する結果が得られます。

## 🛠 使い方

1. **モデル設定**: 梁タイプを選択し、スパン長を入力（カンマ区切りで連続梁に対応）。
2. **断面設定**: 材料種別を選び、断面サイズを指定。
3. **荷重入力**: 荷重の種類・大きさ・位置を入力して「追加」。
4. **結果確認**: グラフや詳細表を確認。必要に応じてCSV出力や印刷を実行。
