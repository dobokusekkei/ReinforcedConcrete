# RC断面計算ツール Ver.6.1

RC（鉄筋コンクリート）部材の断面検討をブラウザ上で行うための計算ツールです。
最大4つの断面を並行して入力・比較できるため、設計の最適化や比較資料作成を効率化します。



---

## 🛠 主な機能

* **4断面一括検討**: 断面寸法の変化や配筋量の違いによる応力度の変化をリアルタイムに比較。
* **1mあたりの断面計算**: スラブや壁などの設計に最適な「単位幅1mあたり」の断面力入力に対応。
* **インタラクティブな描画**: 入力された鉄筋径や寸法に基づき、断面図を動的に描画。
* **印刷最適化レイアウト**: A4縦に全断面を収めるレイアウトで、そのまま設計資料として活用可能。
* **データ保存・読込**: 入力内容をJSON形式でローカルに保存し、いつでも再開可能。

## 📐 計算根拠・主要数式

本ツールは、日本の各種道路・鉄道橋示方書等の「許容応力度設計法」に準拠した計算を行っています。

### 1. 曲げ応力度（軸力を考慮した3次方程式）
中立軸 $x$ の算出には、以下の適合条件および力の釣り合い式から導かれる3次方程式を用いています。

$$x^3 - 3L x^2 - \frac{6n}{b} [A'_s(L-d_p) + A_s(L-d)] x + \frac{6n}{b} [A'_s d_p(L-d_p) + A_s d(L-d)] = 0$$

ここで、 $L = h/2 - M/N$ （偏心距離を考慮した距離）

### 2. せん断応力度（補正係数の適用）
コンクリートの許容せん断応力度 $f_a$ は、部材高および鉄筋比による補正係数を乗じて算出します。

$$f_a = \tau_{a1} \cdot C_e \cdot C_{pt} \cdot \alpha$$

| 補正係数 | 内容 | 算出式 |
| :--- | :--- | :--- |
| **$C_e$** | 部材高補正 | $\sqrt[4]{1000/d}$ (0.7 ～ 1.5) |
| **$C_{pt}$** | 引張鉄筋比補正 | $\sqrt[3]{100 \cdot p_t}$ (0.7 ～ 1.5) |

## 📋 使用上の注意

1.  **入力単位**: 断面力は「1mあたりの値（kN・m/m, kN/m）」として入力してください。
2.  **軸力の符号**: 圧縮力を「正（プラス）」として入力してください。
3.  **適用範囲**: 本ツールは長方形断面を対象としています。円形断面やT形断面の計算には対応していません。
4.  **免責事項**: 計算結果の正確性については万全を期していますが、実設計に使用する際は、必ず設計者自身による確認を行ってください。
