# RC断面計算ツール (4断面一括検討) Ver.6.1

このツールは、鉄道・道路構造物等のRC（鉄筋コンクリート）部材における、曲げ応力度およびせん断応力度を計算するためのWEBアプリケーションです。最大4つの断面を同時に比較・検討でき、設計の最適化や比較資料の作成を効率化します。

---

## 🚀 主な特徴

* **4断面同時比較**: 異なる配筋案や断面寸法を並べて比較検討が可能。
* **リアルタイム計算**: 数値を変更した瞬間に応力度と判定（OK/NG）を更新。
* **設計実務に即した入力**: 断面力は「部材幅1mあたり」の値を入力するスラブ・壁構造向けの仕様。
* **インタラクティブな図解**: 入力された寸法や鉄筋径に基づき、断面図を正確な縮尺で描画。
* **印刷・保存機能**: A4縦1枚に4断面を収める最適化レイアウトでの印刷と、入力データのJSON保存・読込に対応。

---

## 🛠 使用方法

### 1. 断面の設定

* **断面タイトル**: 検討箇所（例：側壁下端、頂版中央など）を入力します。
* **材料条件**: コンクリート強度(Fc)を選択すると、許容応力度が自動設定されます。必要に応じて割増係数（短期荷重時  など）を入力してください。
* **断面形状**: 幅  (通常は1000mm)、高さ 、かぶり  を入力します。

### 2. 配筋の入力

* 鉄筋径と本数を選択します。1mあたりの断面積  および引張鉄筋比  が自動算出されます。

### 3. 断面力の入力
